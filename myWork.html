Sistema de Lista De Espera
Funcionalidades Principais:

• Conexão com Banco de Dados: O sistema se conecta a um banco de dados MySQL, facilitando a consulta e manipulação de dados de clientes e usuários:
• Interface de Usuário: Utilizando PySimpleGUI, criei interfaces intuitivas para adicionar, editar e excluir clientes, assim como adicionar e gerenciar usuários:
• Gerenciamento de Usuários: A interface permite que usuários com diferentes níveis de permissão (master, junior, consultor) realizem ações específicas, garantindo que a segurança e a hierarquia de acesso sejam respeitadas:
• Consulta de Dados: O sistema consulta dados da tabela lista_espera, permitindo a filtragem por modelo de moto e exibindo as informações de forma clara e organizada:
• Validação e Tratamento de Erros: Foram implementadas várias validações e mensagens de erro para melhorar a experiência do usuário e garantir a integridade dos dados:
• Exclusão de Clientes: Somente usuários com permissão master podem excluir registros, o que adiciona uma camada extra de segurança ao sistema:
• Atualmente, estou aprimorando a interface de edição de clientes e buscando otimizar a consulta de dados. Também planejo adicionar gráficos para melhor visualização das informações:
